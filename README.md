# Text-to-Speech Converter

A responsive and interactive web application that converts written text into natural-sounding speech using **HTML**, **CSS**, and **JavaScript**. Users can type text, select from multiple voices, and listen instantly. The interface is clean, intuitive, and works seamlessly across devices.

## Features
- Real-time text-to-speech conversion
- Multiple voice options via browser speech synthesis API
- Clean, visually appealing, and fully responsive design
- Works across modern browsers

## Technologies Used
- **HTML5**: Semantic structure and content  
- **CSS3**: Styling, gradients, and responsive design with media queries  
- **JavaScript**: Dynamic voice selection, speech synthesis, and event handling  

## How to Use
1. Clone the repository or download the project files.
2. Open `index.html` in any modern web browser.
3. Type or paste text into the textarea.
4. Select a voice from the dropdown.
5. Click **Listen** to hear the text spoken aloud.
